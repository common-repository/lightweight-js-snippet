Easy and really lightweight insert script functionality to a specific page.

== Description ==
Go to settings and add the JavaScript code, pick a page, save.

== Installation ==
1. Upload "/lightweight-js-snippet/" to the "/wp-content/plugins/" directory.
1. Activate the plugin through the "Plugins" menu in WordPress.
